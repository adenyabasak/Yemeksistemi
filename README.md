# Food Management System

Food Management System is an ASP.NET Core MVC application developed using Entity Framework Core and the Code First approach.

The project includes a modern home page, an admin panel with CRUD operations, a reporting page, and PDF / Excel export features. A Bootstrap-based admin template was integrated to provide a clean, responsive, and user-friendly management interface.

# Admin Panel

The project includes a Bootstrap-based admin panel.

Admin panel features:

- Category listing
- Category creation
- Category update
- Category deletion
- Recipe listing
- Recipe creation
- Recipe update
- Recipe deletion
- Report page
- PDF export
- Excel export

---

# Reporting Module

The reporting page provides statistical information about the system.

The reports were created using LINQ queries and include count-based statistics and data analysis.

Reports can be exported as:

- PDF
- Excel

---

# Export Features

The project supports report export in two formats.

- PDF export with QuestPDF
- Excel export with ClosedXML
